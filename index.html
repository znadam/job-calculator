🧠 工作性价比计算器算法设计逻辑

核心数学模型：综合年收入和生命时间成本的比率结算。

1. 真实时薪计算公式（融合假期、居家办公与餐饮折算）

本工具创造性地将带薪假天数、居家办公 (WFH) 以及包三餐等隐性财务福利直接融入核心硬性时薪公式，实现底层数据的优雅穿透：

餐饮福利日省金额 = (包早餐? 15 : 0) + (包午餐? 35 : 0) + (包晚餐? 40 : 0)
餐饮福利年省金额 = 餐饮日省 * 实际年工作日
调整后年打包综合年薪 = 原年打包收入 + 餐饮福利年省金额（免税纯省）

实际年工作日 = 250天(标准工作日) - 年度带薪年假天数
折算日均通勤 = 往返通勤时间 * (1 - 每周WFH天数 / 5)
每日生命占用时间 = 每日在岗时间 + 折算日均通勤
年度生命占用总工时 = 实际年工作日 * 每日生命占用时间
核算生命售出时薪 = 调整后年打包综合年薪 / 年度生命占用总工时

*这意味着：餐饮等免税省钱福利会被直接折算进年薪分盘中，大幅拉高你的“生命售出时薪”。

1.5 “纯搬砖付出时薪”与时间挽回逻辑

为了还原打工人最真实的体感，我们推出了“纯搬砖付出时薪”指标：

日挽回个人时间 = 在公司运动洗澡时间 + 摸鱼/处理私事时间 + 餐饮省下的备餐家务时间(每个包餐约0.2-0.3h)
每日纯纯搬砖时间 = 每日在岗时间 - 运动洗澡时间 - 摸鱼时间
纯搬砖付出时薪 = 调整后年打包综合年薪 / (每日纯纯搬砖时间 * 实际年工作日)

*这意味着：如果你在上班时间高强度摸鱼、在公司健身房撸铁、洗好澡再回家，虽然你的生命占用时间没变，但你实际用于劳动搬砖的付出时间被极大压缩。系统会通过“纯搬砖付出时薪”给你一个极其震撼且解气的“真实劳动报酬率”反馈，让你拥有极高的时间自主获得感！

2. 硬性指标得分与“健康惩罚因子”

真实时薪得分根据实际时薪与期望时薪的比率计算，但为了杜绝“低要求、高折损”的算法漏洞，我们引入了生命健康折损惩罚因子（Survival Penalty Multiplier）：

日总工时系数 = 日均工作时间 + 往返总通勤时间
当 日总工时系数 ≤ 9小时：不惩罚（乘数 = 1.0）
当 9小时 < 日总工时系数 ≤ 20小时：启动渐进惩罚，公式为 Math.pow(1 - (日总工时 - 9) / 11, 1.2)
当 日总工时系数 ≥ 20小时：评分强制直接归零（100% 剥夺）

*这意味着：一个日总耗时 12 小时的岗位（即使薪酬达标）由于生理局限，总评分会被健康乘数扣掉大约 30%；16 小时折损系数更恐怖（降至 30% 分数附近）；而达到 20 小时（接近人类生存极限）时，哪怕给再多钱、期望薪资设得再低，总分都将被一票否决降为 0 分，真实反映生命的承载极限。

3. 软性维度加分逻辑 (三大维度，各满分 20 分)

  • 时间掌控权 (20分)：由“上班摸鱼自由(10分)”与“上下班弹性WFH(10分)”之和计算。衡量的是你在名义工作时间里，多大程度上能为自己所用。
  • 精力损耗度 (20分)：由“工作轻松度(10分)”和“下班生活边界(10分)”之和计算。工作如果对精力造成毁灭性榨干，即使下班自由，剩余的时间也没有高质量价值。
  • 保障与未来 (20分)：由“综合保障公积金(10分)”和“履历背书与未来跳槽杠杆(10分)”之和计算。确保打分体系不仅关注眼前，还考虑到对抗未来失业风险的能力。

4. 性价比等级划分

  • 🏆 得分 > 80: “神仙岗位” - 拥有无与伦比的自主时间比例或极高时薪，生活与工作的黄金平衡。
  • 🥈 得分 70-80: “真中产舒适岗” - 整体性价比较佳，无过度内耗，且能保障品质生活。
  • 🥉 得分 60-70: “性价比平平” - 优缺点分明，通常用大量的加班/高强度/高通勤，来换取差强人意的薪水。
  • ⚠️ 得分 < 60: “牛马大军警告” - 时薪低于预期，生命时间被工作严重剥夺，且伴随高强度精神损耗与微弱保障。
